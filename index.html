Call Method

The call() method calls a function functionName with a given this value and arguments. The first argument of the call() method thisArg is the this value. It allows you to set the this value to any given object. The remaining arguments of the call() method arg1 , arg2 ,â€¦ are the arguments of the function.

var person1 = {firstName: 'Bala', lastName: 'naren'};
function say(greeting) {
console.log(greeting + ' ' + this.firstName + ' ' + this.lastName);
}
say.call(person1, 'Hello');

Apply Method

The apply() method is an important method of the function prototype and is used to call other functions with a provided this keyword value and arguments provided in the form of array or an array like object. Array-like objects may refer to NodeList or the arguments object inside a function.

var person1 = {firstName: 'Bala', lastName: 'naren'};
function say(greeting) {
console.log(greeting + ' ' + this.firstName + ' ' + this.lastName);
}
say.apply(person1, ['Hello']);

Bind Method

The bind() method creates a new function, when invoked, has the this sets to a provided value.The bind() method allows an object to borrow a method from another object without making a copy of that method. This is known as function borrowing in JavaScript.

var person1 = {firstName: 'Bala', lastName: 'naren'};
function say() {
console.log('Hello ' + this.firstName + ' ' + this.lastName);
}
var sayHello = say.bind(person1);
sayHello();

IIFE Method

An Immediately-invoked Function Expression (IIFE for friends) is a way to execute functions immediately, as soon as they are created. IIFEs are very useful because they don't pollute the global object, and they are a simple way to isolate variables declarations.

(function () {
console.log('Hello Bala naren')
}) ();

Spread operator

The spread operator allows us to spread the value of an array (or any iterable) across zero or more arguments in a function or elements in an array (or any iterable). The rest parameter allows us to pass an indefinite number of parameters to a function and access them in an array.

let array1 = [1, 2, 3];
let array2 = [...array1];
console.log(array2);

Function sum (num1, num2, num3) {
console.log(`num1 ${num1}, num2 ${num2}, num3 ${num3}`)
console.log(num1 + num2 + num3)
}
const arr = [1, 2, 3]
sum(...arr)

Rest operator

Rest parameter is an improved way to handle function parameter, allowing us to more easily handle various input as parameters in a function. The rest parameter syntax allows us to represent an indefinite number of arguments as an array

function add (...num) {
console.log(num)
}
add(1, 2, 3, 4, 5)
